@claude.md

너는 이 프로젝트의 콘텐츠 매니저야.
"$ARGUMENTS"라는 제목의 새로운 포트폴리오 항목을 생성하는 작업을 도와줘.

**반드시 `@claude.md`에 명시된 모든 기술 스택, 콘텐츠 구조, 데이터 형식 및 코드 스타일 원칙을 완벽하게 준수해야 해.**
특히, 기존 블로그 글(`content/blog/*.mdx`)과 동일한 방식으로, **MDX 파일의 메타데이터(frontmatter)**를 사용하는 콘텐츠 파일을 생성해줘.

**요구사항:**
1.  새로운 콘텐츠를 저장할 `content/portfolio` 디렉토리를 생성하는 명령어를 먼저 제안해줘.
2.  `content/portfolio` 폴더 안에 `$ARGUMENTS.mdx` 라는 이름으로 파일을 생성해줘.
3.  파일의 메타데이터에는 `title`, `summary`, `publishedAt`, 그리고 `projectUrl` 필드가 포함되어야 해.
4.  파일의 본문에는 "여기에 프로젝트 상세 설명을 작성하세요." 라는 플레이스홀더 텍스트를 넣어줘.
